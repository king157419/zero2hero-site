## 1.2 化为标准型

令 `u = -x1 >= 0`，`x3 = v - w`，其中 `v >= 0, w >= 0`，并引入松弛变量 `s >= 0`。

标准型可写为：

```text
min z = -2u - x2 + 2v - 2w

s.t.
u + x2 + v - w = 4
u + x2 - v + w + s = 6
u, x2, v, w, s >= 0
```

若要求极大化标准型，可令 `Z = -z`：

```text
max Z = 2u + x2 - 2v + 2w
```

```text
max w = 15y1 + 20y2 - 5y3

 5y1 - 6y2 - y3 <= -6
-3y1 + 10y2 - y3 = -7

y1 >= 0, y2 <= 0, y3 无约束
```
